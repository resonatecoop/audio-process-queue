# Audio processing

Convert audio files to streamable m4a using fluent-ffmpeg.

## Install

```sh
npm install
```

## Build (babel)

```sh
npm run build
```

## Env

```sh
cp .env.example .env
```

## Usage

```sh
node ./lib/index.js run --name my-queue-name
```

## Test

```sh
npm test
```
